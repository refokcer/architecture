# Lab4

Выполнено 3 отдельных проекта (без вложенных подпапок внутри `Lab4`):

- `Lab4.Structural` — структурный стиль.
- `Lab4.OOP` — объектно-ориентированный стиль.
- `Lab4.Functional` — функциональный стиль.

Все 3 проекта эквивалентны по функционалу:
- пользователь вводит верхнюю границу;
- программа считает сумму квадратов натуральных чисел, которые простые или совершенные.
