# Lab6 — Фрактальні структури і шаблони

Консольний застосунок, який за параметрами користувача будує фрактал і зберігає результат у форматі SVG.

## Підтримувані фрактали
- Трикутник Серпінського
- Килим Серпінського
- Сніжинка Коха

## Шаблони проєктування
- **Strategy (поведінковий):** інтерфейс `IFractalGenerator` + окремі генератори для кожного фрактала.
- **Factory Method / Simple Factory (породжуючий):** `FractalGeneratorFactory` створює потрібну стратегію за типом фрактала.
- **Decorator (структурний):** `MetricsRendererDecorator` обгортає рендерер і додає метрики часу/кількості сегментів.

## SOLID
- **S**: класи мають одну відповідальність (генерація, рендер, UI, оркестрація).
- **O**: додавання нового фрактала не вимагає зміни існуючих генераторів.
- **L**: всі реалізації `IFractalGenerator` взаємозамінні.
- **I**: вузькі інтерфейси `IFractalGenerator`, `ISceneRenderer`, `IFractalGeneratorFactory`.
- **D**: `FractalApplication` залежить від абстракцій, а не від конкретних реалізацій.

## Структура
- `Domain` — моделі, абстракції, алгоритми генерації.
- `Infrastructure` — фабрика та SVG-рендеринг.
- `Application` — use-case рівень.
- `Presentation` — консольний інтерфейс.
