# Звіт до Lab6

## Тема
**«Фрактальні структури і шаблони»**

## Опис рішення
Розроблено консольний застосунок, який дозволяє обрати тип фрактала та параметри генерації (ітерації, розмір полотна, кольори, товщина ліній). Результат візуалізується у файлі SVG.

## Реалізовані фрактали
1. Трикутник Серпінського
2. Килим Серпінського
3. Сніжинка Коха

## Використані шаблони
1. **Породжуючий:** `FractalGeneratorFactory`.
2. **Поведінковий:** `Strategy` (`IFractalGenerator` + конкретні алгоритми).
3. **Структурний:** `Decorator` (`MetricsRendererDecorator`).

## SOLID
- **SRP:** UI, генератори, рендеринг, фабрика і застосунок розділені.
- **OCP:** новий фрактал додається через нову реалізацію `IFractalGenerator`.
- **LSP:** клієнт працює з інтерфейсом `IFractalGenerator`.
- **ISP:** розділені вузькі інтерфейси доменного рівня.
- **DIP:** `FractalApplication` інвертує залежності через інтерфейси.
