# Lab5 — Система управління бібліотекою (SOLID)

Консольний застосунок з мінімальним UI для:
- пошуку книг;
- перевірки наявності;
- оренди книг;
- керування обліковими записами користувачів.

## Запуск
```bash
dotnet run --project Lab5/Lab5.csproj
```

## Структура
- `Domain` — моделі та абстракції
- `Application` — бізнес-сервіси
- `Infrastructure` — in-memory репозиторії
- `Presentation` — консольний UI
- `Documentation` — звіт по SOLID


## SOLID
- Детальна мапа реалізації всіх 5 принципів: `Documentation/Report.md`.
