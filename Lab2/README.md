# Lab2 — Калькулятор (MVC)

Цей проєкт реалізує десктопний застосунок **«Калькулятор»** на Windows Forms із чітким поділом за шаблоном **MVC (Model-View-Controller)**.

## Структура проєкту

- `Models/`
  - `CalculatorModel.cs` — уся бізнес-логіка обчислень, стан калькулятора, обробка операцій.
  - `OperationType.cs` — перелік підтримуваних математичних операцій.
- `Views/`
  - `ICalculatorView.cs` — контракт View (події натискання та метод оновлення дисплея).
  - `CalculatorForm.cs` — графічний інтерфейс (кнопки, дисплей, прив’язка UI-подій).
- `Controllers/`
  - `CalculatorController.cs` — зв’язує View та Model, реагує на події з UI.
- `Program.cs` — композиційний корінь, ініціалізація `Model + View + Controller`.
- `Documentation/Report.md` — звіт і висновки за вимогами лабораторної.

## Реалізовані операції

- Додавання, віднімання, множення, ділення.
- Десяткова крапка.
- Зміна знаку числа (`±`).
- Видалення останнього символу (`⌫`).
- Очищення поточного вводу (`CE`) та повне очищення (`C`).

## Як запустити

```bash
dotnet run --project Lab2/Lab2.csproj
```

> Примітка: проєкт орієнтований на Windows (`net8.0-windows`, WinForms).
