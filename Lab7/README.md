# Lab7 — Текстовий редактор

GUI-застосунок для базової роботи з текстовими документами:
- створення нового документа;
- відкриття/збереження (`.txt`, `.md`);
- операції редагування (`Undo/Redo/Cut/Copy/Paste`);
- меню, панель інструментів і статусний рядок.

## Запуск
```bash
dotnet run --project Lab7/Lab7.csproj
```

## Структура
- `Domain` — модель документа.
- `Application` — абстракції, команди, сценарії редактора.
- `Infrastructure` — робота з файловою системою і фабрика обробників форматів.
- `Presentation` — WinForms UI.
- `Documentation/Report.md` — архітектурний аналіз і патерни.
